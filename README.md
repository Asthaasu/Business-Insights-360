# Business Insights 360 – Data Analytics Project

## Project Overview
Developed an end-to-end Business Intelligence solution for AtliQ Hardware to analyze finance, sales, and executive performance using interactive dashboards.

## Tools & Technologies
- Tableau
- SQL
- Excel
- Data Modeling

## Key Features
- Built 3 interactive dashboards: Finance, Sales, Executive
- Integrated data from multiple sources (Excel/CSV & SQL)
- Designed KPI tracking for revenue, profit, orders and customers
- Created dynamic filters for market, platform and fiscal year
- Performed trend and category performance analysis

## Business Impact
- Analyzed ₹1M+ transactional data across multiple markets
- Identified top 10 customers contributing ~30% revenue
- Improved reporting efficiency by ~60% using interactive dashboards
- Enabled data-driven decision making for sales & finance teams
